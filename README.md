# SQL Data Pipeline (DWH)

## Описание проекта
Проект представляет собой реализацию ETL-процессов в хранилище данных (DWH) на базе PostgreSQL.  
Основная цель — автоматизация загрузки, трансформации и проверки качества данных при формировании буферного слоя.

## Функциональность
- Динамическое создание буферных таблиц на основе метаданных  
- Загрузка данных из исходных представлений во временные (buffer) таблицы  
- Формирование структуры таблиц через генерацию SQL-запросов  
- Логирование этапов выполнения ETL-процесса  
- Проверка качества данных (DQ):
  - Выявление дубликатов  
  - Удаление дубликатов с использованием оконных функций (`row_number()`)

## Технологии
- PostgreSQL  
- PL/pgSQL  

## Структура
- `f_dm_core_2_buf.sql` — функция создания и загрузки буферных таблиц  
- `f_dm_dq_buf_check.sql` — функция проверки качества данных и удаления дубликатов  

## Результат
Реализован универсальный механизм формирования буферного слоя и контроля качества данных, позволяющий автоматизировать ETL-процессы и снизить количество ошибок в данных.
